很久之前就想搭建属于自己的个人博客，为什么是很久之前，真相只有一个：懒！于是拖到了19年的春节，选择在春节搭建博客主要是能够打（tai）发（wu）时（liao）间（le），有益身心。大学是仿佛看到过用github+hexo搭建博客的文章，于是下定决心玩一下这两个玩意。

github的简单操作：

1.注册一个个人github账号（很简单）

我的github主页（欢迎访问）：https://github.com/DataZwer

2.创建一个个人仓库（Repository）

注意名字用的是自己的用户名+github.io,如本文的仓库名为datazwer.github.io。
先放着，接下来会用到。

git下载路径： https://git-scm.com/downloads

hexo是一个简单高效的博客框架，我们可以通过hexo搭建出许多个性化的博客页面。而node.js是一个让 JavaScript 运行在服务端的开发平台，它让 JavaScript 成为与PHP、Python、Perl、Ruby 等服务端语言平起平坐的脚本语。

2.初始化博客

在文件系统中创建一个空文件夹，命名随意，如：myblog，右键选择Git Bash Here，输入命令：

hexo init

这就是一套标准的博客模板。

3.安装必要的组件

npm install

4.体验Hexo

开启服务：

hexo s

按提示访问本地版博客，打开浏览器，访问localhost:4000/，即可访问到博客的默认模板(后面后介绍一个很漂亮的模板)：
